# SQLite를 이용한 Django로 게시판 만들기

## 장고 설치

```
pip install django
```

## 프로젝트 시작

```
django-admin startproject [project name]
```

## 앱 만들기

```
python manage.py startapp 앱이름
```

## 서버 실행

```
python manage.py runserver
```
